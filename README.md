# Đạo Sinh Tâm - Kinh Dịch AI

Ứng dụng web huyền học tích hợp AI, giúp bạn khám phá 64 quẻ Kinh Dịch, gieo quẻ, và nhận được lời giải thích từ trí tuệ cổ xưa.

## 🌟 Tính Năng

- **64 Quẻ Kinh Dịch** - Khám phá chi tiết 64 quẻ với ý nghĩa, giải thích và lời khuyên
- **Quay Quẻ Ngẫu Nhiên** - Gieo quẻ với vòng quay tương tác và nhận lời giải thích
- **Chat AI Thông Thái** - Hỏi AI để nhận lời giải thích sâu sắc về Kinh Dịch
- **Tìm Kiếm Internet** - Chat AI tích hợp tìm kiếm web để cung cấp thông tin mới nhất
- **Lịch Sử Hội Thoại** - Lưu trữ và xem lại các cuộc hội thoại trước
- **Kết Nối Mạng Xã Hội** - Theo dõi trên Facebook, TikTok và các kênh khác

## 🛠️ Công Nghệ

- **Frontend**: React 19 + TypeScript + Tailwind CSS 4
- **Backend**: Express.js + tRPC 11
- **Database**: TiDB (MySQL-compatible)
- **AI**: LLM Integration + Web Search API
- **Auth**: Manus OAuth

## 📦 Cài Đặt

```bash
# Cài đặt dependencies
pnpm install

# Chạy dev server
pnpm dev

# Chạy tests
pnpm test

# Build production
pnpm build
```
